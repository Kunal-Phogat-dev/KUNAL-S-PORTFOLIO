**Goal:** Build a high-converting personal portfolio that attracts international clients and showcases execution speed, modern skills, and AI integration. Target: First $500 in 30 days.

## 1. Overall Vision & Feel

- **Vibe:** Bold, confident, high-energy but professional. "Young Musk + Naval Ravikant" energy — ambitious, no-BS, results-obsessed.
- **Dark mode first** with smooth light mode toggle.
- **Performance:** Load under 1.5 seconds. Lighthouse score ≥ 95.
- **Feel for User:** Premium yet approachable. Visitor should instantly think: "This guy ships fast and delivers results."
- **Mobile-first:** Buttery smooth on all devices.
- **Color Scheme:** Deep black / dark navy background, neon cyan/blue accents, white/gray text.
- **Typography:** Inter or Satoshi (modern sans-serif). Bold headings, good readability.

## 2. Tech Stack (Non-Negotiable)

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui + Aceternity UI
- **Animations:** Framer Motion
- **AI Integration:** Gemini API for chat avatar
- **Hosting:** Vercel
- **Forms:** Formspree or Resend
- **Other:** TypeScript (recommended), Next/Image for optimization, Lenis for smooth scroll
